# 引言
- 抑郁症高发
- 传统治疗方法：药物、心理，效果不够
- 近年来物理治疗应用于临床，效果较好，如rTMS
- rTMS原理，传统治疗模式的局限，需要额外开发快速有效的治疗模式
- θ突发刺激（TBS）是近年来提出的一种新型rTMS，原理，与传统方法的对比
- TBS的两种类型：连续和间歇，研究发现iTBS 的增强作用强于 cTBS 的抑制作用，且 iTBS 的效果变化更大，目前应用于抑郁症的情况
- 此前的研究表明，抑郁和焦虑之间以及大脑相关区域的变化存在显着相关性，尤其是前额皮质（PFC）
- fNIRS的原理、优势、应用
- 先前的研究表明，青少年抑郁症患者经常表现出异常的大脑激活模式和较低的任务相关功能连接（FC）。与健康对照组相比，重度抑郁症患者在言语流利性任务（VFT）中氧合血红蛋白的增加有所减少。
# 目的
本研究拟通过刺激传统刺激部位DLPFC，结合fNIRS设备，探讨iTBS模式对患者焦虑抑郁的治疗效果。可进一步证实f NIRS在疾病诊断及疗效评价方面的优势。也可以为今后探索更有效的刺激参数提供实践依据。

# 研究对象
所有患者均来自2020年9月至2021年12月华中科技大学同济医学院附属同济医院住院部，共37例。
## 纳入标准：
1. 患者符合《精神障碍诊断与统计手册V》中关于焦虑抑郁障碍的诊断标准
2. 医院焦虑抑郁量表中焦虑、抑郁得分均高于7分。其中女性24例，男性13例，平均年龄19.68 ± 2.56岁。
## 排除标准：
1. 急性脑出血及急性感染性疾病患者。
2. 有癫痫病史及家族史。
3. 有强烈自杀倾向的抑郁症患者。
4. 严重头痛、高血压、恶性肿瘤、开放性伤口、血管栓塞、白细胞减少等患者。入组时Had - a：M ± SD = 8.78 ± 4.09，Had-D：M ± SD = 10.76 ± 3.90。

# 方法
## 刺激参数
采用深圳盈之科技生产的M - 100型极限脉冲磁场刺激仪，线圈型号为BY90A。r TMS刺激部位均为左侧DLPFC。iTBS模式下的刺激参数为：刺激强度为30 % MT，刺激频率为50 Hz，每个单位间隔8 s发放30个包含内部高频( 50 Hz )的短脉冲；刺激次数s为600个脉冲/串；字符串间的间隔时间为8 s，共20个字符串；总刺激时间为3 min 20 s；刺激脉冲总数为12 000个。治疗期间由治疗师对每位入组患者进行rTMS治疗，并留存安全性评估记录。
## 评估
所有患者分别于干预前和干预10次后进行量表和VFT评估。采用医院焦虑抑郁量表(齐格蒙德和Snaith , 1983)评价患者的抑郁和焦虑症状；通过VFT过程中血氧信号的变化评估脑激活情况。block包含30 s的任务前休息、60 s的VFT任务和60 s的任务后休息。
## 感兴趣区域 (ROI)
在脑网络分析中，空间分离的大脑区域与动态生理变化的时间过程之间的相关性称为FC。 FC旨在探讨大脑区域之间是否存在功能耦合关系，以解释大脑活动的潜在机制。分析时间序列中每个通道中氧气和血红蛋白浓度的皮尔逊相关系数，以估计 ROI 之间的成对相关强度。计算功能连接强度以观察ROI之间的功能网络连接。 ROI 设置在以下区域：左 SMC、右 SMC、FE、FL、左 DLPFC、右 DLPFC、左布罗卡区、右布罗卡区、FP。
## 数据处理
Matlab用于处理fNIRS数据。数据处理步骤如下：筛选数据→Intensity2OD→运动伪影校正→滤波→OD2Hb。由于OxyHb比DeoxyHb具有更优越的信噪比并且能够更直接地反映任务相关的皮质激活。因此，我们通过确定治疗前后平均 OxyHb 值的差异来计算 53 个通道中 OxyHb 的平均激活变化。采用SPSS20.0统计软件对数据进行统计处理，采用配对样本t检验检验患者的抑郁、焦虑症状及血氧水平。 α = 0.05（双侧）表示显着性水平，p &lt; 0.05 表示统计显着性。

# 讨论
## iTBS治疗的有效性
本研究结果显示，患者治疗前后焦虑、抑郁评分差异显着。后测成绩明显低于前测成绩。因此，焦虑抑郁障碍患者的焦虑抑郁症状在接受iTBS治疗后得到明显缓解。这一结果与之前的研究结果一致，支持了 iTBS 模式在个体焦虑和抑郁治疗中的有效性（Bulteau et al., 2022）。与传统 rTMS 中高频所达到的效果类似，iTBS 模式诱导神经功能兴奋性的长期增加（LTP-长期增强效应）（Amer 和 Martin，2022）。 TBS模式的特点是集群刺激，更接近神经活动的生理状态，刺激强度比传统rTMS弱。同时，TMS模式的刺激大大缩短了治疗时间，提高了使用效率。
## 近红外成像中的DLPFC改变
许多研究表明，抑郁症患者存在PFC功能障碍，并且与健康对照相比，精神障碍患者在认知VFT过程中脑血流量减少（Zhang等，2015）。本研究利用近红外光谱仪探讨 iTBS 模式刺激对焦虑抑郁患者左侧 DLPFC 的抗焦虑、抗抑郁作用。**结果显示，iTBS治疗后患者大脑布罗卡区和额叶的血流动力学明显高于治疗前。即患者相关认知任务中相关脑区的激活在治疗后增加，血流量的增加导致神经元突触的激活。** 这些结果也与之前的研究结果一致（Huang et al., 2022）。对于抑郁症常出现的PFC激活异常，经过iTBS干预后可得到明显改善。 rTMS的这种高爆发频率可以提高这种激活水平并调节PFC相关的情绪功能。对干预前后不同脑区之间的功能连接进行进一步的相关分析。**结果显示，抑郁症的变化与左前额叶背外侧叶氧合血红蛋白的变化呈显着负相关**。这一发现进一步证明抑郁症与前额皮质的功能有关，前额皮质被认为是调节情绪和认知的大脑区域。额叶皮层活动减少会导致个体抑郁和精神压力。然而，TMS刺激PFC改善了由于左侧DLPFC激活不足而导致的抑郁和悲伤状态。
## 在Broca’s area的改变
我们发现iTBS患者在干预后布罗卡区的激活也显着增加。抑郁症患者通常表现出左额叶激活程度较低，无法在休息和语言任务中很好地激活相关区域（布罗卡区）（Spironelli et al., 2020）。先前的研究人员证明，在布罗卡区进行rTMS干预7天后，DC值（全脑中一个体素与其他体素之间的连接数量）显着增加并逐渐恢复到正常水平（Rao等人， 2021）。因此，iTBS 可能成为语言和认知大脑区域神经可塑性的潜在工具。然而，之前报道的一些结果表明，这些区域的氧血红蛋白在治疗后有所下降。可能有以下几个原因：
- 一是**刺激部位**不同。有研究表明，刺激右侧DLPFC具有抑制作用，同时联合双侧刺激也有不一致的结果。因此，刺激部位很可能会影响治疗的结果，这是我们稍后将继续讨论的部分。然而，本研究中的刺激部位是左侧 DLPFC。在以往的研究中，关于其治疗效果的争议很小，几乎都是积极的作用。
- 其次，可能与**刺激参数**有关。这部分研究目前还比较少，也是未来需要研究的一个方向。本研究刺激参数的有效性也为今后的治疗提供了很好的参考。最后是治疗时间。有研究人员发现，长期（半个月到一个月）治疗前后并无差异，但患者反映在治疗过程中确实感受到情绪明显改善。阅读大量研究后，有一种新的可能性。 iTBS 可能有一个最佳治疗时间段。治疗效果有点类似“倒U型”图，一定时间后效果并不明显。
综上所述，大量研究指出传统的rTMS或iTBS可以改善布罗卡区和额叶的血流动力学，但少数一致性较差的原因有待进一步探讨。此后，iTBS也被应用于中风后失语症，并证实了iTBS与语言功能恢复之间的相关性(Xu et al., 2021)。具体来说，iTBS不仅有助于改善患者的焦虑、抑郁和认知功能，还有助于改善患者的语言能力。目前的研究结果也发现了这种对 iTBS 的治疗作用。
## FC的改变
本研究表明焦虑抑郁症患者大脑的FC在iTBS模式的干预下得到增强。干预前受试者所有ROI的FC强度也显着低于干预后。抑郁症的发作不仅涉及单个区域，而且涉及整个大脑网络。此前的研究发现抑郁症患者的边际奖励回路存在异常（Ng et al., 2019），导致临床患者难以感到快乐。瑞安等人。 (2022)证明他的PFC的iTBS调节奖励过程，并且这种增加与rmPFC-VS的连接性有关，这证明了iTBS对大脑网络的调节。其他研究人员表明，iTBS干预后患者的腹侧PFC和右顶下叶灌注迅速增加，这意味着个体大脑的激活水平在iTBS刺激下得到了提高；他们还指出，左侧DLPFC刺激可以直接影响左右DLPFC与左侧海马的功能和结构连接区域之间的神经元活动（Wu et al., 2022）。这种增强的连通性进一步改变了患者的记忆和认知功能。有研究人员对边缘下PFC神经元的激活标志物进行定量统计，以确定iTBS的治疗效果。与治疗后第3、5、7天的大鼠相比，发现第1天大鼠的细胞数量显着增加，这表明iTBS可以恢复神经元活性。抑郁症患者的海马体往往较小，而海马体是记忆形成的重要大脑区域。 PTSD患者接受iTBS治疗后的神经影像技术显示，默认模式网络更强（更大的正）连接性和负相关（更大的负）跨网络连接性也解释了iTBS治疗的机制；它通过增强大脑活动和功能区域之间的连接来改善患者的各种执行功能和情绪（Philip et al., 2019）。

# 局限性
在这项研究中，与言语相关的大脑区域的血氧水平显着提高。然而，没有使用相关的语言任务来检查个体的语言功能。这种变化需要通过直接语言任务来进一步衡量，这可以成为未来工作的进一步方向。

# 总结
本研究结果表明，iTBS对于治疗焦虑和抑郁症有效，个体的焦虑和抑郁症状在干预后得到显着改善。通过近红外光谱仪对干预前后患者大脑血氧水平的检测显示，干预后参与者大脑区域的血氧水平明显升高，激活水平也有所提高。结果还表明，iTBS 的干预可以增强大脑区域之间的连接性。
